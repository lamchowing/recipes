# Ingredients

- #雞翼 10隻 (約 900克)
- #洋蔥 1大個 （切粗塊）
- #泰國青咖喱醬 (green curry paste) 50克
- #椰奶 (coconut milk) 1杯
- #雞上湯 1/2杯
- #乾青檸葉 (lime leaves) 3片 （稍為撕開）
- #蒜頭 2粒 （拍扁）
- #鹽 適量
- #糖 適量
## Marinade

* #生抽 （醬油） 2湯匙
- #糖 1/2茶匙
- #鹽 1/8茶匙
- #粟粉 （玉米粉）1茶匙
- #紹興酒 2茶匙
- #胡椒粉 少許
# Directions

1. 先把雞翼洗凈，瀝乾水份，用醃料醃 3個小時。
2. 燒熱油鑊，把雞翼的表面煎至微黃。不需要熟透。兜起備用。
3. 再燒熱油鑊，爆香蒜頭，洋蔥，加青檸葉，然後放下青咖喱醬爆透。聞到很香的咖喱味，就倒進雞上湯煮滾，放回雞翼，用小火炆所有材料至軟身，約 15分鐘。
4. 注入椰奶。自己要試味，加適量的鹽，糖調味，上碟。趁熱供食。
# Remarks
1. 對某些人來說，譬如基絲汀，50克青咖喱感到有點辣。但嗜辣的人士並不覺得一回事。基絲汀的寶貝公主，就覺得剛剛好。未嘗試吃過青咖喱的話，一定要試試，的確很好味喔！
