# Ingredients

- #香茅 1枝
- #檸檬葉 4塊
- #乾蔥 3g
- #米酒 2湯匙
- #豬頸肉 2件
- #蒜頭 4瓣
- #黃糖 2湯匙
- #老抽 2湯匙
- #蜜糖 適量
- #魚露 2湯匙
- #青檸汁 1茶匙
- #芫茜 1束
## Source

* #水 1湯匙
* #指天椒 1隻
* #青檸汁 2湯匙
* #黃糖 1茶匙
* #魚露 2湯匙
# Directions

1. 預熱焗爐至攝氏200度，焗盤鋪上錫紙備用。
2. 將香茅、蒜頭、檸檬葉、乾蔥及芫茜切碎備用。
3. 豬頸肉用老抽、魚露、米酒、黃糖、青檸汁、香茅、蒜頭、檸檬葉、乾蔥及芫茜醃起碼1小時。
4. 放上焗盤，放入焗爐焗8分鐘，反轉另一面再焗8分鐘。
5. 掃上蜜糖再焗2-3分鐘，切片上碟。
6. 將魚露、青檸汁、指天椒、黃糖及清水拌勻，伴以豬頸肉享用。
